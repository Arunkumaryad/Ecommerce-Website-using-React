## Run Locally

Clone the project

```bash
  //yha pe github ka dalna hai.
```

Go to the project directory

```bash
  cd React_E-Commerce
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

* [React](https://reactjs.org/)
